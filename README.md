# 🎉 Event Organizer Website

A modern and responsive Event Organizer website built using HTML, CSS, and JavaScript. This website helps users plan, promote, and manage events with an engaging and user-friendly interface.

---

# 🚀 Features

✅ Responsive Homepage  
✅ Event Planning Section  
✅ Event Promotion Cards  
✅ Event Management UI  
✅ Contact Form  
✅ Upcoming Events Showcase  
✅ Mobile Responsive Design
